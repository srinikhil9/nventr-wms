2

3

Done

Your IDE agent reads the rules automatically. No config, no runtime.

Quick Start

pip install agentrun

All Commands

agentrun install <pkg> [pkg2...]   Install packages (registry, GitHub, or local)
agentrun install user/repo         Install from GitHub repo
agentrun install ./local/path      Install from local directory
agentrun uninstall <pkg>           Remove a package and its files
agentrun update [pkg]              Update one or all packages
agentrun list                      Show installed packages
agentrun search <query>            Search the registry
agentrun info <pkg>                Show package details
agentrun init                      Initialize agentrun in your project
agentrun validate ./path           Validate a package before publishing
agentrun publish ./path            Get publish instructions

Creating a Package

A package is a directory with package.yaml, a rules/ folder of .mdc files, and an optional agents.md.

my-rules/
  package.yaml       # metadata
  rules/
    my-rule.mdc      # Cursor MDC format
  agents.md          # cross-IDE content

package.yaml

name: my-rules
version: 1.0.0
description: "My custom coding rules"
author: your-name
tags: [custom]
rules:
  - file: rules/my-rule.mdc
    activation: auto           # always | glob | auto | manual
agents_md:
  section: "My Rules"
  file: agents.md

.mdc Rule Format

---
description: My coding guidelines
globs: "**/*.py"
alwaysApply: false
---
# My Rules

- Rule instruction one
- Rule instruction two

How Rules Are Applied

Cursor uses four activation modes, controlled by the .mdc frontmatter:

alwaysApplied to every chat session (alwaysApply: true)
globApplied when matching files are open (globs: "**/*.py")
autoAI decides based on the rule description
manualOnly when explicitly @-mentioned in chat

AGENTS.md sections are always visible to any IDE agent that reads the file (Copilot, Claude Code, Windsurf, Devin, etc.).

Dependencies

Packages can declare dependencies on other packages:

requires: [python-expert]

Dependencies are installed automatically. Circular dependencies are detected and rejected.
